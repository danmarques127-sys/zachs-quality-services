# Security Policy

## Supported Versions
This project is currently maintained as a static website.
Security updates are applied on a best-effort basis.

## Reporting a Vulnerability
If you discover a security vulnerability, please report it responsibly.

📧 Contact: dev@zachsqualityservices.com  
(or open a private GitHub Security Advisory if available)

Please include:
- A clear description of the issue
- Steps to reproduce (if applicable)
- Potential impact

Do not disclose vulnerabilities publicly until they have been reviewed.

Thank you for helping keep this project secure.
